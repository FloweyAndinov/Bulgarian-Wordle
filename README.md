<br/>

<h1>Set up a local development server</h1>

<ul>
<li> Clone the repo </li>
  
```
git clone https://github.com/FloweyAndinov/Bulgarian-WordBomb.git
```

<li> Host the server</li>

```
cd BW-BackEnd
npm i
node app.js
```
<li>visit at </li>

[localhost:5000](http://localhost:5000) 


</ul>
